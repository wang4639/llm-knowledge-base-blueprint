# 知识库使用手册

## 日常使用

直接向 AI 提问即可，但要求它先阅读 [[AI Agent 使用指南]]。AI 应先命中板块和问答卡，再决定是否需要来源笔记。

## 添加资料

1. 把 Markdown 原件放进本地配置指定的资料目录；
2. 运行 `scripts/Check-Inbox.ps1`；
3. 检查五项差异；
4. 只有专用摄入任务且安全门禁通过时才继续；
5. 摄入后再次检查并运行动态验证。

## 什么不会自动发生

- 不会删除、移动、改名或覆盖原件；
- 不会把 AI 的看法写成使用者本人观点；
- 不会因文件名相同自动判断重复；
- 不会把新资料静默覆盖旧结论；
- 不会把整个来源层当作每次问答的默认上下文。

## 人与 Agent 的分工

- Agent：结构化、编译、链接、检查和报告；
- 使用者：授权资料、确定板块、填写个人观点、确认 evergreen、裁决冲突和高风险变更。
